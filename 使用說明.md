# YouTube 音訊下載器 - 使用說明

## 🎯 最簡單的使用方式

### 方法一：雙擊執行（推薦！）

1. **找到檔案**：`下載YouTube音訊.bat`
2. **雙擊執行**
3. **貼上 YouTube 連結**，按 Enter
4. **完成！** 音訊檔案會儲存在 `downloads` 資料夾

就這麼簡單！

### 方法二：PowerShell 執行

打開 PowerShell，切換到此目錄後執行：

```powershell
cd c:\Users\admin\Desktop\youtube-to-audio
npm.cmd run interactive
```

## 📝 使用步驟詳解

1. **執行程式** - 雙擊 `下載YouTube音訊.bat`
   
2. **貼上連結** - 複製 YouTube 影片連結並貼上
   ```
   請貼上 YouTube 連結: https://www.youtube.com/watch?v=xxxxx
   ```

3. **選擇格式** (可選，直接按 Enter 使用 MP3)
   ```
   選擇格式 (1=MP3, 2=M4A, 3=WAV) [預設: 1]:
   ```

4. **選擇輸出位置** (可選，直接按 Enter 儲存到 downloads)
   ```
   輸出目錄 [預設: c:\...\downloads]:
   ```

5. **等待下載完成** - 程式會顯示下載進度

6. **繼續或離開**
   - 按 `Y` 或 Enter：繼續下載其他影片
   - 按 `N`：結束程式

## 📂 檔案位置

下載的音訊檔案預設儲存在：
```
c:\Users\admin\Desktop\youtube-to-audio\downloads\
```

檔案名稱會自動使用 YouTube 影片的標題。

## 💡 提示

- 可以連續下載多個影片，無需重新啟動程式
- 支援的格式：MP3（推薦）、M4A、WAV
- 自動提取最佳音質
- 如果想離開程式，輸入 `q` 即可

## ⚠️ 注意事項

- 首次使用可能需要一些時間下載
- 請確保網路連線正常
- 僅下載您有權使用的內容
- 遵守 YouTube 服務條款

## 🔧 進階使用（命令行模式）

如果您想使用命令行模式，可以執行：

```powershell
npm.cmd run dev "YouTube連結" -o "輸出目錄" -f mp3
```

但大多數情況下，直接雙擊 `下載YouTube音訊.bat` 就夠用了！
